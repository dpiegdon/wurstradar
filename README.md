
STM32F429Discovery Pinout
=========================

Mögliches Pinout bei verwendung des Disco-Boards:

PA5: DAC  Channel 2
PC3: ADC1 Channel 13
PF6: ADC3 Channel 4

(siehe `wurstradar_stm32f429_discovery.ioc`)
